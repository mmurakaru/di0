# examples - the workspace template

Generic sample content: a SaaS-subscriptions profile (`di0.profile.yml`), an
ad-hoc query in `queries/`, a deliverable bundle in `arr_overview/` (spec + its
queries), and an agent-context note in `context/`. Nothing here is real - it's a
starting shape.

`di0 init` copies this into your gitignored `workspace/`. Replace it with your own
queries, profiles, and bundles. See [../docs/workspace.md](../docs/workspace.md).
