# examples - the workspace template

Generic sample content: a SaaS-subscriptions profile (`di0.profile.yml`), sample
`queries/`, a dashboard spec in `deliverables/`, and an agent-context note in
`context/`. Nothing here is real - it's a starting shape.

`di0 init` copies this into your gitignored `workspace/`. Replace it with your own
queries, profiles, and specs. See [../docs/workspace.md](../docs/workspace.md).
