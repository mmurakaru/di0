# narrate

A method for keeping initiative context next to an analysis without hard-coding it
anywhere reusable. The numbers come from the other skills; this skill supplies the
"why" and the framing.

## Method

1. State the initiative and the decision in one or two sentences.
2. Pull any genuinely non-derivable context from the agent-context note
   (`context/`), not from code.
3. Frame the validated findings against that context.

## Rule

Narrative belongs in versioned context notes and deliverables, never in the core
or in scattered query literals. This skill names no warehouse.
